DEVELOPMENT COMMANDS:
- Start Laravel dev server: php artisan serve (runs on http://localhost:8000)
- Build assets for development: npm run dev
- Watch for asset changes: npm run dev (add --watch)
- Build for production: npm run build
- Create new migration: php artisan make:migration migration_name
- Create new controller: php artisan make:controller ControllerName
- Create new model: php artisan make:model ModelName
- Run migrations: php artisan migrate
- Tinker (REPL): php artisan tinker

DATABASE CONFIGURATION:
Update .env file with your database credentials:
- DB_CONNECTION (mysql, pgsql, sqlite, etc.)
- DB_HOST (localhost)
- DB_PORT (3306)
- DB_DATABASE (your_database)
- DB_USERNAME (root or your user)
- DB_PASSWORD (your password)

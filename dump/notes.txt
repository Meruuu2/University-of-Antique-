QUICK START COMMANDS
================================================================================

1. START DEVELOPMENT SERVER:
   php artisan serve
   → Access application at http://localhost:8000

2. BUILD FRONTEND ASSETS:
   npm run dev    (development mode with hot reload)
   npm run build  (production build)

3. DATABASE SETUP (update .env first):
   php artisan migrate

4. CREATE NEW BLADE VIEW:
   Create file in resources/views/yourview.blade.php
   Route to it in routes/web.php

5. CREATE NEW CONTROLLER:
   php artisan make:controller YourController

6. CREATE NEW MODEL WITH MIGRATION:
   php artisan make:model YourModel -m

7. RUN TESTS:
   php artisan test
   vendor/bin/phpunit

8. INTERACTIVE SHELL (Tinker):
   php artisan tinker
